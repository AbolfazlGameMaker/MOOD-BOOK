# Mood Book

**What this program does:**  
Mood Book is a personal journaling application that analyzes user notes to determine their emotional state.  
It allows users to:  

- Write personal notes or diary entries  
- Detect emotions such as Happy, Sad, Angry, Anxious, or Neutral  
- Track emotional trends over time with charts  
- Store notes securely in a database for future reference  

In short, it provides **a simple and user-friendly way to record, analyze, and visualize personal emotions**.

---

## ‚ö° Features

- Easy-to-use text editor for writing notes  
- Automatic sentiment analysis with multiple emotions  
- Save and load notes from a secure database  
- Visual charts to track emotional trends over time  
- Color-coded notes based on detected emotion
